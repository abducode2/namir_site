# نمير للتطوير العقاري والمقاولات

موقع إلكتروني احترافي لشركة نمير، المتخصصة في التطوير العقاري والمقاولات في المملكة العربية السعودية.

## المميزات

- ✅ تصميم عصري وجذاب بالكامل
- ✅ وضع ليلي / نهاري (Dark/Light Mode)
- ✅ تصميم متجاوب بالكامل (Responsive)
- ✅ تأثيرات حركية عند التمرير (AOS)
- ✅ نموذج اتصال تفاعلي
- ✅ أيقونات جميلة من Font Awesome
- ✅ تحسين لمحركات البحث (SEO)

## التقنيات المستخدمة

- React.js 18
- CSS Modules / CSS Variables
- AOS (Animate on Scroll)
- Font Awesome 6
- Google Fonts

## كيفية التشغيل

1. قم بتثبيت الاعتماديات:

```bash
npm install
```
